stat

The stat function retrives the information about the specified directory.

int sysinfo
(
   const string& path,
   vector<SNode>& attr
);
Parameters
path
[in] the name of the directory.
attr
[out] a list of SNode attributes for every file or directory in the directory.
Return Value

On success, the functions returns 0. If error occurs, a negative value is returned.

Error Name Comment
E_INVALID This Sector instance has not been initialized
E_NOEXIST path does not exist
E_PERMISSION no read permission on the path
