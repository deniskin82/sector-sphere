init

The init function initializes the Sphere stream .

int init
(
   const vector<string>& files
);


int init
(
   const int & num
);
Parameters
files
[in] The list of files that form the input stream
num
[in] The number of output files
Return Value

On success, the function returns the number of files in the stream. If num is negative, the second format of the function returns 0. On error, a negative value is returned.

Error Name Comment
E_RESOURCE No enough system resource (memory) to complete this request
Description

The first init function only records the input data but does not query the Sector file system to check the input files, which is done in run or run_mr. If there is any input files are invalid, the error will be returned from run or run_mr.
