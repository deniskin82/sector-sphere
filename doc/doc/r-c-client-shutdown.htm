Reference: Classes: Sector

shutdown

The shutdown function can be used to shutdown part of or the complete Sector system.

int shutdown
(
   const string& type,
   const string& param = ""
);
Parameters
type
[in] type = "a": shutdown whole system, no "param" required; type = "d", shutdown one slave, "param" is the ip:port address; type = "i", shutdown one slave, "param" is slave ID.
param
[in] shutdown parameter.
Return Value

On success, the function returns a unique session key assigned by the Master. If error occurs, a negative value is returned:

Error Name Comment
E_INVALID this Sector instance has not been initialized
E_AUTHORITY no authority to run this command
E_CONNECTION connot connect to the master
Description

The shutdown function requires "root" user.
