mkdir

The mkdir function creates a new directory in the Sector system.

int remove
(
   const string& path
);
Parameters
path
[in] the name of a file or directory to be created.
Return Value

On success, the function returns 0. If error occurs, a negative value is returned.

Error Name Comment
E_INVALID This Sector instance is not initialized
E_CONNECTION cannot connect to the master
E_PERMISSION no permission for IO
E_EXIST directory already exists
