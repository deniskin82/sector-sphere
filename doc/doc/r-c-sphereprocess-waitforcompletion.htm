waitForCompletion

The waitForCompletion function blocks the current thread until the Sphere process is completed.

int checkProgress
(
);
Parameters

None.

Return Value

On success, the function returns 0. If error occurs, a negative value is returned.

Error Name Comment
E_NOPROCESS No Sphere process is running
E_ALLSPEFAIL No active SPE is available
E_BUCKETFAIL At least one bucket process is failed, cannot recover
Description

This function should be used to wait for the Sphere to completed finished to start further processing. If no result is sent back from the SPEs (i.e., reults are either sent to buckets or written into slave local files), the client program can simply call waitForCompletion after run or run_mr to wait the processing to finish.
