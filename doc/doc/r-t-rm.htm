Reference: Tools

sector_rm

The sector_rm tool deletes a file or a directory.

sector_rm PATH [--f]
Parameters
PATH
The path to the file or the directory to be deleted.
Description

If PATH is a directory and it is not empty, the rm tool will ask for user permission to remove all files in the directory recursively, unless the [--f] option is set, in which case all directories will be deleted silently.

There is no way to restore the deleted files. To remove files without deleting them permanently, you may use sector_mv to move such files into a "rubbish" directory.
