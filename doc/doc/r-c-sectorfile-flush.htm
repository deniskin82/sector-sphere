flush

The flush function clears all unwritten data in the local buffer.

int flush
(
);
Parameters
localpath
[in] the local file name to store the downloaded file.
cont
[in] if the local file already exists, the function may continue to download from the offset of the local file size (assuming it is a broken download from previous try) or overwrite the local file.
Return Value

On success, the function returns size of data downloaded. If error occurs, a negative value is returned.

Error Name Comment
E_FILENOTOPEN This file has not been opened
E_CONNECTION Connections to slaves have been lost
E_LOCALFILE Local file system error, e.g., disk full
Description

The download function does not change the current read offset position.
